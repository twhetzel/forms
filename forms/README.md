# README

Directory of forms to test options for autocomplete of text fields. Tests include jQuery and typeahead.js/Bloodhound.js for use with forms generated using Alpaca Forms(http://www.alpacajs.org/).

# Dependencies
Bootstrap 3.3.2
jQuery 1.11.1
Handlebars 3.0.3
Alpaca 1.5.14
Bloodhound 
typeahead 0.10.5
